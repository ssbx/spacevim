Spacevim and XTerm configs
==========================


### install

```shell
$ rm -rf ~/.SpaceVim
$ git clone git@github.com:ssbx/spacevim.git ~/.SpaceVim
$ cd ~/.SpaceVim
$ make install
```

#### python

```shell
$ sudo apt install pylint yapf3 autoflake python3-isort
$ npm install -g pyright
```
